sio2pc

I made an sio2pc using an FTDI Basic Breakout from Sparkfun. Make sure to use the 5V one instead of the 3.3V one. I ran aspeqt on my PC. The sio2pc allows one to run disk images from the PC on the Atari and save files from the Atari to the PC.
